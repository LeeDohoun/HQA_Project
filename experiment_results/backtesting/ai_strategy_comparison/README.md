# AI Strategy Comparison

Start here:

- `AI_STRATEGY_COMPARISON_REPORT.md`: human-readable conclusion and comparison tables.
- `comparison_table/`: combined multi-agent centered comparison table.
- `technical_baseline_runs_2023_2026q1/`: RSI, Bollinger Band, momentum, and volatility-adjusted momentum baseline runs.
- `source_multi_agent_runs/`: raw short/long multi-agent and deterministic run outputs used by the comparison.
- `source_comparison_build/`: raw technical baseline runs used when building the multi-agent centered comparison.

This folder is intentionally narrow. It keeps only artifacts related to the current question: whether short/long project multi-agent hybrid strategies are useful compared with deterministic and simple technical strategies.
